# Parsers

Parsers are the core building blocks of TagScript that process tags and return values based on the tag syntax. You can use built-in parsers or create your own custom parsers.

## Usage

```typescript
import { Interpreter, RandomParser, RangeParser, FiftyFiftyParser, IfStatementParser, SliceParser } from 'tagscript';

const ts = new Interpreter(
  new SliceParser(),
  new FiftyFiftyParser(),
  new RandomParser(),
  new IfStatementParser()
);
```

## Custom Parsers

You can create custom parsers by implementing the `IParser` interface:

```typescript
import { BaseParser, type Context, type IParser } from 'tagscript';

export class FetchParser extends BaseParser implements IParser {
  public constructor() {
    super(['fetch'], false, true);
  }

  public parse(ctx: Context) {
    return fetch(ctx.tag.payload!.trim()).then((res) => res.text());
  }
}
```

## Parser Categories

### Control Flow
- [IfStatementParser](/parsers/if-statement) - Conditional logic with comparison operators
- [ControlParser](/parsers/control) - Union and intersection statement parsing
- [BreakParser](/parsers/break) - Force specific output based on conditions
- [StopParser](/parsers/stop) - Halt all processing

### Randomization
- [RandomParser](/parsers/random) - Select random items from lists
- [FiftyFiftyParser](/parsers/fifty-fifty) - 50/50 probability outcomes
- [RangeParser](/parsers/range) - Generate random numbers in ranges

### String Manipulation
- [SliceParser](/parsers/slice) - Extract substrings
- [ReplaceParser](/parsers/replace) - Replace text patterns
- [FormatParser](/parsers/format) - Format text case and escape characters
- [UrlEncodedParser](/parsers/url-encoded) - URL encoding/decoding

### Variables & Data
- [DefineParser](/parsers/define) - Define custom variables
- [LooseVarsParser](/parsers/loose-vars) - Access variables with loose checking
- [StrictVarsParser](/parsers/strict-vars) - Access variables with strict checking
- [JSONVarParser](/parsers/json-var) - Parse and store JSON data

### Utilities
- [IncludesParser](/parsers/includes) - Check text inclusion and find indices
