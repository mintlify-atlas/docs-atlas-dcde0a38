## What is TagScript?

TagScript is a drop-in, easy-to-use string interpreter that lets you provide users with ways of customizing their profiles or chat rooms with interactive text. It's a powerful template engine that processes special tags in strings and transforms them into dynamic content.

Think of it as a safe, user-facing scripting language where users can write templates like:

```typescript
{random:Alice,Bob,Charlie} won the game!
I pick {if({5050:.}!=):heads|tails}
```

And TagScript will process these tags to generate dynamic output.

## Why TagScript?

TagScript was designed to solve a common problem: letting users customize text content without giving them access to dangerous operations or exposing your application to security risks.

## Key Features

<AccordionGroup>
  <Accordion title="Full TypeScript Support" icon="code">

    - **Randomization**: `{random}`, `{range}`, `{5050}` for random content
    - **Variables**: `{=}`, `{var}` for storing and reusing values

    - And many more!

## Quick Example

Here's a taste of what TagScript can do:

```typescript
import { Interpreter, RandomParser, IfStatementParser, FiftyFiftyParser } from 'tagscript';

const ts = new Interpreter(
  new RandomParser(),
  new IfStatementParser(),
  new FiftyFiftyParser()
);

const template = `
  {random:Parbez,Rkn,Priyansh} attempts to pick the lock!
  I pick {if({5050:.}!=):heads|tails}
`;

const result = await ts.run(template);
console.log(result.body);
// Output: "Parbez attempts to pick the lock! I pick heads"
// or: "Rkn attempts to pick the lock! I pick tails"
```

## Common Use Cases

TagScript is perfect for:

- **Discord bots** - Custom commands, welcome messages, and auto-responses
- **Chat applications** - User-customizable messages and profiles
- **Content management** - Template-based content generation
- **Game servers** - Dynamic announcements and player-facing scripting
- **Custom commands** - Let users create their own command responses
